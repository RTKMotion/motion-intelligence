# Changelog

Tracks notable updates to the RTK Motion MCP server presence across catalogs and registries. The hosted server itself runs at `https://api.rtkmotion.io/mcp` on Cloudflare Workers; this changelog documents the publishing history.

## 2026-04-29

- **Repository created**: this manifest repo published at `github.com/RTKMotion/motion-intelligence` under the new `RTKMotion` GitHub organization. Public, MIT-licensed for repo contents (data products separately licensed via x402).
- **Smithery listing live**: server published at `smithery.ai/servers/RTKMotion/motion-intelligence`, verified via `www.rtkmotion.io` TXT record + URL backlink, all 14 tools auto-detected.
- **MCP Server Registry v1.2.0 published**: `io.rtkmotion/data` updated from v1.1.0 to v1.2.0 at `registry.modelcontextprotocol.io`. v1.2.0 broadens the title to cover all three verticals (motion capture, biomechanics, threat intelligence), adds `websiteUrl`, and expands keywords. v1.0.0 (2026-03-14) and v1.1.0 (2026-04-02) remain in registry history.
- **Glama**: positioned for auto-discovery via the `glama.json` in this repo + GitHub topic tags (`mcp`, `model-context-protocol`, etc.). Awaiting their crawler.

## 2026-04-02

- **MCP Server Registry v1.1.0**: `io.rtkmotion/data` updated.

## 2026-03-14

- **MCP Server Registry v1.0.0**: `io.rtkmotion/data` initial publish to `registry.modelcontextprotocol.io`.
